# Spotit Clinical Safety Case Template

## Scope
Spotit supports wound documentation, wound photo capture, structured assessment, care-plan recording, and escalation prompts.

Spotit must not be positioned as replacing clinician judgement or diagnosing wound conditions.

## Intended Users
- Nurses
- Support workers
- Nurse assistants
- Tissue viability nurses
- Duty clinicians
- Doctors

## Key Hazards
- Wound assigned to the wrong patient
- Wound photo stored against wrong wound/site
- Incorrect measurement entered
- Escalation prompt missed
- Inappropriate over-reliance on automated prompt
- Photo consent missing
- Report sent to wrong recipient
- User accesses records without permission

## Controls
- Patient identity confirmation
- Photo consent prompt
- Audit log
- Mandatory photo/measurement before wound assessment
- Clinician confirmation before care-plan completion
- Clear wording that prompts do not diagnose
- Role permissions
- Report destination confirmation

## Evidence Needed
- Usability testing
- Clinical review of prompts
- Escalation rule validation
- Penetration test
- Data protection review
- Pilot incident log
