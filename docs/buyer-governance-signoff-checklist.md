## Required Checks

- [ ] Intended use confirmed
- [ ] Demo reviewed with fake/test data
- [ ] DPIA completed
- [ ] GDPR lawful basis confirmed
- [ ] Data Processing Agreement agreed
- [ ] Privacy notice approved
- [ ] Data retention period agreed
- [ ] Data deletion process agreed
- [ ] Incident response process agreed
- [ ] Access control policy approved
- [ ] User onboarding/offboarding process approved
- [ ] Backup and restore process reviewed
- [ ] Wound photo storage reviewed
- [ ] Audit log process reviewed
- [ ] Clinical safety review completed
- [ ] Regulatory/medical-device review completed where applicable
- [ ] Staff training completed
- [ ] Local SOP approved
- [ ] Pilot risk assessment completed
- [ ] Support contact agreed

## Decision

- [ ] Approved for demo only
- [ ] Approved for controlled pilot with fake/test data
- [ ] Approved for controlled pilot with real patient data
- [ ] Not approved
