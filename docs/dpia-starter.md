# Spotit DPIA Starter

## Data Processed
- Patient identifiers
- Date of birth
- NHS number or local patient number
- Allergies
- Past medical history
- Wound history
- Wound photos
- Wound measurements
- Dressing and care-plan details
- Clinician notes
- Audit logs

## Special Category Data
Spotit processes health data and wound images. These are special category data under UK GDPR.

## Main Privacy Risks
- Unauthorised access
- Wrong patient selected
- Photo shared or exported incorrectly
- Data retained longer than necessary
- Weak password or shared account
- Insecure device use

## Required Controls
- Secure named-user login
- Role-based permissions
- Encrypted data storage
- Private image storage
- Audit logging
- Session timeout
- Data retention policy
- Backup and recovery policy
- Incident response process
- User training

## Open Decisions
- Hosting location
- Data controller and processor roles
- Retention period
- Report sending method
- Customer support access rules
- Live patient data pilot approval
